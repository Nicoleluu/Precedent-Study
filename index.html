Measuring Fairness
Nicole Lu
Abstract

Measuring Fairness is an interactive web explorable developed by Google PAIR that helps users understand how fairness in machine learning can be measured in multiple ways. Through visual simulations and adjustable parameters, the project demonstrates how fairness metrics can conflict and why designing fair AI systems is complex.
