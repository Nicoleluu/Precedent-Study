Data Is Never Neutral

Inspired by Google PAIR's Measuring Fairness and Hidden Bias, the poster shifts attention from the algorithm to the data itself, asking viewers to reconsider where bias begins.

What the Viewer Discovers
01

A Claim

DATA IS NEVER NEUTRAL. Every dataset reflects decisions about what is included, excluded, categorized, and remembered.

02

A Constructed Word

Up close, the letters are built from categorized words. What initially appeared unified and objective is revealed as an accumulation of selected data.

03

Partial Understanding

Each language represents a different category, not a translation of the same vocabulary. Viewers understand only the languages they can read, so each person receives a different part of the poster’s context. This limited access shapes interpretation and sharpens the claim that bias begins with partial knowledge.

The Dataset Inside “Neutral”

The word field contains nine distinct categories, each expressed through a different language and a different number of terms. The languages are not translations of one shared vocabulary. Because viewers can read only some of them, the context available to each viewer changes. The uneven quantities also produce different densities within the letters and across the background.
